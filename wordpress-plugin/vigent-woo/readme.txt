افزونهٔ اتصال ووکامرس به ویجنت — محصولات و سفارش‌ها را به ایجنت هوش مصنوعی می‌فرستد.

== Description ==

این افزونه محصولات و سفارش‌های ووکامرس شما را به پنل ویجنت می‌فرستد تا ایجنت هوش مصنوعی بتواند
به سؤال‌های مشتری دربارهٔ قیمت، موجودی و وضعیت سفارش پاسخ دقیق بدهد. هر تغییری در محصول یا سفارش
به‌صورت خودکار به webhook ویجنت ارسال می‌شود و دکمهٔ هم‌گام‌سازی کامل دستی هم در صفحهٔ تنظیمات قرار دارد.

= امکانات =

* ارسال خودکار رویدادهای ساخت، ویرایش و حذف محصول
* ارسال خودکار رویدادهای تغییر وضعیت سفارش
* دکمهٔ هم‌گام‌سازی کامل دستی برای همهٔ محصولات یا سفارش‌ها
* امضای HMAC-SHA256 برای امنیت webhook
* صفحهٔ تنظیمات فارسی زیر منوی WooCommerce

== Installation ==

= نصب افزونه =

1. پوشهٔ `vigent-woo` را در مسیر `wp-content/plugins/` کپی کنید (یا فایل zip را از طریق
   صفحهٔ «افزونه‌ها → افزودن → بارگذاری افزونه» نصب کنید).
2. به صفحهٔ «افزونه‌ها» بروید و افزونهٔ «ویجنت — اتصال ووکامرس» را فعال کنید.

= دریافت آدرس webhook از پنل ویجنت =

1. وارد پنل ویجنت (https://vigent.ir) شوید.
2. به صفحهٔ «اتصالات» → بخش «فروشگاه آنلاین» بروید.
3. روی «افزودن فروشگاه» کلیک کنید، نوع را «WooCommerce» انتخاب کنید و آدرس فروشگاه خود را وارد کنید.
4. پس از ذخیره، یک آدرس webhook به‌همراه کلید امنیتی به شما نمایش داده می‌شود. هر دو را کپی کنید.

= تنظیم کلید امنیتی =

1. در پیشخوان وردپرس به مسیر «WooCommerce → ویجنت» بروید.
2. آدرس webhook را در فیلد «آدرس webhook ویجنت» جای‌گذاری کنید.
3. کلید امنیتی را در فیلد «کلید امنیتی (webhook secret)» وارد کنید.
4. تیک هم‌گام‌سازی محصولات و سفارش‌ها را بزنید و روی «ذخیره تنظیمات» کلیک کنید.

= فعال‌سازی هم‌گام‌سازی محصولات و سفارش‌ها =

* برای هم‌گام‌سازی خودکار، کافی است تیک گزینه‌های مربوطه روشن باشد. از این پس هر تغییر در محصول
  یا سفارش به‌صورت لحظه‌ای به ویجنت ارسال می‌شود.
* برای هم‌گام‌سازی کامل اولیه یا بازسازی داده‌ها، از دکمه‌های «ارسال همهٔ محصولات» و «ارسال همهٔ
  سفارش‌ها» در پایین صفحهٔ تنظیمات استفاده کنید.

== Frequently Asked Questions ==

= آیا نیاز به REST API ووکامرس داریم؟ =

خیر. این افزونه از هوک‌های PHP ووکامرس استفاده می‌کند و داده‌ها را از طریق webhook به ویجنت
می‌فرستد. نیازی به فعال‌سازی کلید REST API نیست.

= آیا اطلاعات مشتری هم ارسال می‌شود؟ =

بله، برای هر سفارش نام، شماره تماس و ایمیل صورت‌حساب ارسال می‌شود تا ویجنت بتواند سفارش را
به مشتری شناسایی‌شده در پنل CRM پیوند بزند.

= چه زمانی رویداد ارسال می‌شود؟ =

* ساخت، ویرایش یا حذف محصول
* تغییر وضعیت سفارش (مثلاً از «در حال انجام» به «تکمیل شده»)
